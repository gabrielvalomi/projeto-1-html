Oque são tags

(voltar)

As tags são os elementos utilizados para indicar ao navegador oque aquilo representa, por exemplo a tag p ira indicar para o navegador que tudo que estiver envolvido dentro desta tag ele deve interpretar como um paragrafo de texto, assim como a tag h1 significa titulo principal.

Anatomia

E de grande inportancia destacar que as tags tem sua anatomia por exempro a tags de abertura e fechamneto como < p > e < / p> sem fazer este processo os navegadores não iram entender oque vc esta querendo dizer.

cliente

Quando sitamos cliente se referimos ao usuario final que seria a pessoa fisica utilizando seu notbook ou seu smartfone para acessar web sites, normalmente utilizando um sistema operacional windowns, ao navegar na intenet o navegador deste usuario ira usar a internet apenas como meio de transporte paar buscar informações dos sites no servidor e banco de dados, isto é todo o codigo criado por um dev esta arquivado em algum servidor apenas esperando ser buscado pelo usuario final, ao usuario difitar o endereço na web seu navegador ira viajar na velocidade da luz ate o servidor e ira coletra as informaçoes deste site. os clientes estar conectador por aparelhos pessoais como celulares notbook tvs smarts entre outros aparelhos, resumidamente os cleints sao todos aqueles que estar usando windonws em aparelhos remotos.

servidor

Servidores, basicamente sao maquinas robustas que como o proprio nome ja indica ira servir o cliente como dispositivo de armazenamento de web site, essas maquinas robustas são encontradas na google, microsoft entre outros locais, tambem podendo ser criada como por exemplo uma grande empresa pode instalar um grande computador central que ira servir dispositivos ao seu redor, este computador central deve conter um sistema operacional de server como CentOS, Debian, Unix, Windows Server entre outrso sistemas, sabendo disto podemos afirmas que a internet e apenas uma porte que liga claint e servers.
